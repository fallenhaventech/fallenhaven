HTML Cheatsheet

HTML Elements and Structures


<a> Anchor Element

Anchor Element defines a link to a website, image, location etc

Text Link:
<a href="http://website.extension"> Related Text </a>
<a href="http://website.extension"> <img src="logo.jpg"> Related Text </a>

<head> Head Element

Head Element defines the metadata content not displayed on
the page, page title, links to stylesheets etc

<!DOCTYPE HTML>
<html>
 <head>
  <!-- Metadata is contained here -->
 </head>
</html>

<target> Target Atribute

Target specifies where the hyperlink will open

<a href="https://www.website.ext" target="_blank">

Will open this site in a new tab</a>

HTML Forms


<form> Form Element

Form element is used to collect and send information to an external source. Can contain various input elements.

Declaration:
<form method="post" action="http://server">
Input Your Name Please?
<input type="text" name="fname">

<input> Types of input

Html input elements /p>

<input type="number" name="balance" />
<input type="text" name="textfield" />
<input type="password" name="pwd" />

Submitting a Form

After collecting the information we have to submit to somewhere else. Uses the action and submit.

<form action="/index3.html" method="PUT"> </form>

HTML Tables


<table> Table Element

Table used to represent a bidimensional table

Table declaration:
<table>
  <!-- rows and columns go here -->
</table>

<tr>Table Row
<td>Table Data
<th>Table Heading

tr is used do add rows to the table, th adds some table headings and finally td adds content/data to the cells


<table>
  <tr>
  <td>
  </td>
  </tr>
</table>

<colspan>Column Span
<rowspan>Row Span

Colspan indicates over how many colums the given data field should span, and rowspan does the same, but over a number of rows
